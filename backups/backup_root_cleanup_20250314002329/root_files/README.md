# LiqPro 项目

LiqPro 是一个基于 Solana 区块链的 AI 驱动投资平台，专注于 Meteora DLMM 流动性池。

## 目录结构

```
LiqPro/
├── production/           # 生产环境配置和部署文件
├── src/                  # 源代码
├── docs/                 # 文档
├── scripts/              # 脚本
├── deploy/               # 部署配置
├── tests/                # 测试
└── .env-files/           # 环境变量配置
```
